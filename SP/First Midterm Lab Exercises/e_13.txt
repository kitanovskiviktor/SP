Полу-елка се црта со ѕвездички (*) и прачки (|) на следниот начин:

*
|
|**
|
|***
|
|****

Значи, во првиот ред има ѕвездичка. Во следниот ред има прачка. Понатаму преку еден ред имаме или само прачка или прачка со неколку ѕвездички (една повеќе отколку во прет-претходниот ред). Полу-елката завршува со ред во кој има една прачка и N ѕвездички.

За дадено N, отпечатете ја полу-елката според описот.

За N=4, треба на излез да се добие како примерот погоре.

#include <stdio.h>

int main() {
    int N;
    scanf("%d", &N);
    int brojac=0;
    int Z=N*2-1;
    
    for(int i=0; i<Z; i++) {
        
        if(i==0) {
            printf("*\n");
            continue;
        }
        if(i%2==1) {
            printf("|\n");
        }
        if(i%2==0) {
            brojac++;
            printf("|");
            for(int j=0; j<=brojac; j++) {
                printf("*");
            }
            printf("\n");
        }
    }
    return 0;
}
